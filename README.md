# Guessie 🐱 - LVJ Soft Tech Test

A beautifully designed number guessing game built with React Native and Expo. 

## ⚡️ Tech Stack

- **Framework**: [React Native](https://reactnative.dev/) & [Expo](https://expo.dev/) (Expo Router)
- **UI Components**: [HeroUI Native](https://v3.heroui.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with [Uniwind](https://uniwind.dev/)
- **Animations**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/) & [Lottie React Native](https://github.com/lottie-react-native/lottie-react-native)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)

## 🎯 Key Highlights

- **Interactive Gameplay**: Engaging number guessing mechanics with real-time feedback.
- **Lottie Animations**: Features an animated cat character that provides visual hints.
- **Smooth Transitions**: animated number counters and dynamic text using Reanimated.
- **Authentication System**: Complete login and registration flows.
- **Onboarding Experience**: Welcoming introductory screens designed to guide new users.

## 🚀 Getting Started

Install dependencies:
```sh
npm install
```

Run on platforms:
```sh
npx expo run:ios     # iOS simulator
npx expo run:android  # Android emulator
```

## ⚠️ Note on Screen Recording

For privacy and security reasons, the password input field automatically obscures characters and hides the on-screen keyboard during screen recording. Therefore, this will not be visible in the demo video.
